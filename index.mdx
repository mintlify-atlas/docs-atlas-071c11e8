Electron lets you create native desktop applications for Windows, macOS, and Linux using web technologies you already know. It embeds Chromium for rendering and Node.js for system access, so you can use any npm package and access native APIs alongside your HTML/CSS/JS frontend.

## Build your first app

<Steps>
  <Step title="Install Electron">
    Add Electron as a dev dependency to your project.

    ```bash
    npm install electron --save-dev
    ```
  </Step>
  <Step title="Create your main process file">
    Create `main.js` to manage your app's lifecycle and windows.

    ```javascript main.js
    const { app, BrowserWindow } = require('electron')

    function createWindow() {
      const win = new BrowserWindow({ width: 800, height: 600 })
      win.loadFile('index.html')
    }

    app.whenReady().then(createWindow)
    ```
  </Step>
  <Step title="Add your HTML page">
    Create `index.html` as your renderer entry point.

    ```html index.html
    <!DOCTYPE html>
    <html>
      <head><title>My Electron App</title></head>
      <body><h1>Hello from Electron!</h1></body>
    </html>
    ```
  </Step>
  <Step title="Run it">
    Launch your app with the `electron` command.

    ```bash
    npx electron .
    ```
  </Step>
</Steps>
